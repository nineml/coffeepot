Chapter 1. Installing CoffeePot

The simplest way to install CoffeePot is to download the latest release zip file from https://github.com/nineml/coffeepot/releases/latest. Unzipping that somewhere on your system will result in a directory that contains a jar file and a few ancillary files. You can run CoffeePot directly from the jar file.

You can also get CoffeePot from Maven with the following coordinates:

  |<dependency>
  |  <groupId>org.nineml</groupId>
  |  <artifactId>coffeepot</artifactId>
  |  <version>3.0.0j</version>
  |</dependency>

For more details about incorporating CoffeePot into a project of your own, see the HOWTO repository.
